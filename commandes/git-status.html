git status

Il s'agit de la commande principale permettant de connaitre l'état des fichiers dans le dépot Git.

Syntaxe

git status

Affichage

$ nano a.txt
$ git status On branch master Untracked files: (use "git add file..." to include in what will be committed) a.txt nothing added to commit but untracked files present (use "git add" to track)

git status -u

Tout comme git status, cette commande indique l'état des fichiers dans le dépot Git, ainsi que les fichiers non suivis. On peut utiliser un certain nombre de paramètres (optionel, par défaut sur all) : no, normal ou all

Syntaxe

git status -u[paramètre]

Affichage

nano a.txt git status -uno On branch master nothing to commit (use -u to show untracked files)

git status -b

Tout comme git status, cette commande indique l'état des fichiers dans le dépot Git, ainsi que les branches du dépot (crées par les commit).

Syntaxe

git status -b

Affichage

git status -b On branch gh-pages
Untracked files:
(use "git add file..." to include in what will be committed)
a.txt
commandes/git-status.html
nothing added to commit but untracked files present (use "git add" to track)
